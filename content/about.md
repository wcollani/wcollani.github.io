This blog is where I publish what I learn from running real systems: benchmarks, architectural patterns, postmortems, and lessons from building agents that actually go wrong in interesting ways.

**Projects:**

- [llm-eval-kit](https://github.com/wcollani/llm-eval-kit) — declarative framework for benchmarking local LLMs against operational tasks

**Stack:** Python · Ollama · Unraid · UniFi · Prometheus · Loki
